HR Analytics Dashboard using Looker Studio
📌 Project Overview

This project is an HR Analytics Dashboard created using Google Looker Studio and Excel to analyze employee data and generate meaningful HR insights.

The dashboard helps visualize:

Employee distribution
Salary analysis
Attendance trends
Performance scores
Gender-based insights
Department-wise analysis
🛠️ Tools Used
Google Looker Studio
Microsoft Excel
📊 Dashboard Features
Total Employee KPI
Average Salary KPI
Average Attendance KPI
Highest Performance Score
Salary by Department
Employee Distribution by Department
Performance Score by Employee
Performance Score by Gender
Interactive Filters
🔍 Key Insights
IT department has the highest average salary.
Sales department has the highest employee count.
Employee attendance is consistently high.
Gender distribution is balanced.
Top-performing employees can be identified easily.
